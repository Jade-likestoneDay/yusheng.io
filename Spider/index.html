docker命令

docker search 用于从 Docker Hub 查找镜像

docker images 显示镜像

docker pull NAME 下载镜像，NAME是用docker search寻找到的镜像名

docker rmi（rm） REPOSITORY/IMAGE ID 一次性删除一个或多个镜像（空格分开）

docker save REPOSITORY/IMAGE ID >（或者使用 -o） hint.tar 将镜像转为tar 导出的时候要加上版本号，否则可能会造成名字丢失

docker ps 命令可以用来列出 Docker容器 相关信息

docker upload -i FILE 导入文件，-i可以选择导入的文件位置。

docker tag NAME（IMAGE ID） NAME 镜像重命名

docker inspect 查看一个或多个镜像详细信息

docker history 查看镜像历史信息

docker create [OPTIONS] IMAGE [COMMAND] [ARG…] 创建一个容器但不启动它

docker start 启动一个或多个处于停止中的容器 -a : 在前台运行容器并输出日志 -i：容器标准输入

docker run == docker create+ docker start -a 创建一个容器并运行一个命令
